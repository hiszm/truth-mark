# 红绿灯系统

TruthMark 在 AI 回答结束后，追加一个整体可信度评估块：

| 总体评级 | 标记 | 含义 |
|:--------:|:----:|------|
| 可信 | 🟢 | 回答基于确定事实、官方文档、数学公理等 |
| AI 推断 | 🟡 | 回答包含推断、建议、趋势预测等 |
| 不可信 | 🔴 | 回答包含超出知识范围、无法验证、实时信息等内容 |

## 评估块格式

```markdown
---
🚦 TruthMark 评估
总体：🔴 红灯 · 不可信
引用："2025 年下半年，GPT-6 将会发布，这将彻底改变市场格局。"
依据：该内容涉及未来产品发布日期，超出我的知识范围，无法验证。
```

## 设计原则

- **不修改原回答**：AI 的原始回答保持原样。
- **整体评估**：根据回答中最严重的问题给出总体评级。
- **引用原文**：黄灯和红灯必须引用原回答中的具体句子。
