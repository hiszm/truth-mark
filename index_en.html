The Traffic Light System

🟢

Green · Verified Fact

Deterministic knowledge from training data, official docs, axioms. Safe to use.

🟡

Yellow · AI Inference

Context-based derivation, suggestion, best practice. Look twice.

🔴

Red · Uncertain

Beyond knowledge cutoff, inaccessible files, real-time info. Do not trust blindly.

One-Line Install

Run this in your terminal on macOS or Linux:

curl -fsSL https://raw.githubusercontent.com/hiszm/truth-mark/main/scripts/install.sh | bash

Key Features

  • Zero configuration after install — every AI response is auto-marked
  • Three-level traffic lights require no learning curve
  • Hooks inject rules automatically, transparent to users
  • Post-tool verification prevents hallucinations from empty results
  • Open source under MIT — free to use and extend
