# Example: Debugging Help

User: What does this error mean?

Error: `ImportError: No module named 'requests'`

Expected response:

🚦
🟢 `ImportError: No module named 'requests'` means Python cannot find the `requests` package.
🟡 You may need to run `pip install requests`.
🟡 If you are using a virtual environment, make sure it is activated.
🔴 I do not know your Python version or operating system.
